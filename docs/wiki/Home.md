# Bayside Hub Wiki

Welcome to the Bayside Hub wiki.

Bayside Hub is a role-aware school operations platform built with Next.js 14 (App Router) and Supabase. It centralizes classes, clubs, events, announcements, internal tools, and administrative workflows in one interface.
